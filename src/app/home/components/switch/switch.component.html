Switch

Switch are used to choose true or false.

Radio

Radios are for selecting one option from many.

Stacked

Disabled radios are supported, but to provide a "not-allowed" cursor on hover of the parent <label> , you'll need to add the .disabled class to the parent .radio, .radio-inline.

Inline

Use the .radio-inline classes on a series of radios for controls that appear on the same line.

Checkbox

Checkboxes are for selecting one or several options in a list.

Stacked

Disabled checkboxes are supported, but to provide a "not-allowed" cursor on hover of the parent <label> , you'll need to add the .disabled class to the parent .checkbox, .checkbox-inline.

Inline

Use the .checkbox-inline classes on a series of radios for controls that appear on the same line.

Style

You'll add class .checkbox-success , .checkbox-primary , .checkbox-info , .checkbox-warning , .checkbox-danger to parent .checkbox .
